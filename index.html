Política de Privacidad

Esta Política de Privacidad describe el tratamiento de datos en un sistema de automatización del hogar basado en Home Assistant, de uso interno y personal.

1. Alcance del sistema

Este sistema de automatización del hogar está diseñado para uso privado dentro de un entorno doméstico. No está destinado a servicios públicos ni comerciales.

2. Plataforma utilizada

El sistema utiliza Home Assistant como plataforma principal de automatización del hogar, ejecutándose en infraestructura controlada por el usuario.

3. Integración con Google Home

El sistema puede integrarse con Google Home para permitir el control por voz de dispositivos y automatizaciones.

Estas integraciones pueden depender de servicios proporcionados por Google para el procesamiento de comandos de voz y ejecución de acciones.

4. Datos procesados

Dependiendo de la configuración, el sistema puede procesar:

  • Estados de dispositivos (luces, sensores, enchufes, etc.)
  • Eventos de automatización
  • Comandos de voz enviados a través de Google Home
  • Registros internos de funcionamiento (logs)

No se recopilan datos personales con fines comerciales ni de seguimiento publicitario.

5. Uso de la información

Los datos se utilizan únicamente para:

  • Automatización del hogar
  • Control de dispositivos internos
  • Mejora del funcionamiento del sistema

6. Almacenamiento de datos

Los datos se almacenan localmente en el entorno del usuario donde se ejecuta Home Assistant, salvo aquellos procesados por servicios externos como Google Home.

7. Servicios de terceros

Algunas funcionalidades pueden depender de servicios de terceros como Google. Estos servicios están sujetos a sus propias políticas de privacidad.

8. Compartición de datos

No se comparten datos con terceros, excepto cuando es necesario para la integración con servicios externos configurados por el usuario.

9. Seguridad

Se aplican medidas razonables de seguridad en el entorno local. Sin embargo, la seguridad depende de la configuración del propio usuario.

10. Cambios en la política

Esta política puede actualizarse si cambian las integraciones o la configuración del sistema.

11. Contacto

Este sistema es de uso personal. Para consultas, contactar al administrador del sistema doméstico.
